### `data/`
This folder contains the datasets used in the project at different stages of the analysis lifecycle.  
- `RAW FILE(Female Political Inclusion in Legislative Elective Positions in South Western Nigeria (2015 - 2023) (Responses).xlsx` holds the original survey data as collected from the Google Form and is preserved for reference and reproducibility.  
- `female_analysis_CLEANED_FILE.xlsx` contains the processed and standardized dataset after data cleaning, transformation, and restructuring, and is the primary source used for analysis.
